How to use this template

Copy this file to the correct subdirectory, rename it, and replace every placeholder. Do not change the section order or heading id attributes — they are referenced from the hub index and sidebar.

File location and naming

  • Folder: services/portal/internal/governance/audits/docs/ for documentation / DX / UI-UX assessments; services/portal/internal/governance/audits/api/ for REST / HTTP API assessments. Do not put new assessments in services/portal/internal/governance/audits/ root.
  • File name: YYYY-MM-DD-topic-assessment.html — ISO date first, kebab-case topic, then -assessment.html.
    Example: 2026-04-23-ui-ux-assessment.html.
  • Title alignment: put the same formation date in <title>, <h1>, Overview, and the first History row.
  • After creating: add a row to the matching stream index and update its History.
