§6Authorization & permissions

API key per shared auth; learner identity from the body. Ownership rules: referenced UUIDs must be owned by the resolved learner; cross-tenant references collapse to 404 ERR_404.

§7Request contract

Headers

Field Type Required Notes
X-API-Key string required Tenant API key.
Idempotency-Key string required See shared rules. Length 1–128 chars.
Content-Type string required Must be application/json.
Accept-Language string optional BCP-47 language tag (e.g. en-US) — drives localised error messages. Default: tenant locale.

Body

Field Type Required Notes
system_user_id string required External user identifier.
system_uuid string · uuid required External tenant UUID.
message string required 1–2000 chars. Trimmed; whitespace-only rejected. PII class: High.
conspectus_uuid string · uuid optional Optional FK to a conspectus owned by the same learner. On delete: SET NULL.
review_log_id integer · int64 optional Optional FK to a specific review log row, when the error is captured during a review session.
visibility enum required private · unlisted · public
