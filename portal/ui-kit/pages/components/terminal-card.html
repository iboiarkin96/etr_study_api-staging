Terminal card

Self-typing curl/bash terminal hero card. Dark surface (theme-agnostic), animated "live" badge, syntax-coloured response, Copy button, and Try in OpenAPI Explorer link. Used in both portal landing pages.

Example

Markup

One element, no inner markup. The component builds the chrome, body, and footer in JS, then runs an IntersectionObserver-driven typewriter.

<div data-component="terminal-card"
data-try-href="/services/portal/internal/api/index.html"
      data-try-label="Try in OpenAPI Explorer"></div>

Attributes

AttributeDefaultPurpose
data-try-href/services/portal/internal/api/index.htmlDestination for the "Try in…" CTA in the footer.
data-try-labelTry in OpenAPI ExplorerLabel text of the CTA. Set per portal if needed.
data-no-copy—Presence suppresses the Copy button.
data-delay320Milliseconds before typing begins after the card enters the viewport.
data-script—id of an inline <script type="application/json"> with custom segments. Each segment is {cls?, text}.

Custom script

Override the default curl call by passing your own ordered segments. Each segment is { cls?: token-class, text: string }.

Token classes

  • docs-terminal__t-mute — punctuation, prompts, separators
  • docs-terminal__t-cmd — command name (curl, jq)
  • docs-terminal__t-url — URL
  • docs-terminal__t-flag — CLI flag
  • docs-terminal__t-str / docs-terminal__t-var — string / variable
  • docs-terminal__t-ok / docs-terminal__t-num / docs-terminal__t-lat — response values

Use in portals

Both portal landings should mount the same component — only the data-try-href differs.

Internal portal

<div data-component="terminal-card"
data-try-href="/services/portal/internal/api/index.html"
      data-try-label="Try in OpenAPI Explorer"></div>

Public portal

<div data-component="terminal-card"
data-try-href="/services/portal/public/reference/api/index.html"
      data-try-label="Try in OpenAPI Explorer"></div>

The mount happens automatically — mountTerminalCard() is wired into both runtime/internal/entry.js and runtime/public/entry.js.

A11y

  • The typewriter respects prefers-reduced-motion — it renders the full script instantly instead of typing.
  • Copy button has aria-label="Copy command" and announces "Copied" on success via the visible label change.
  • The CTA link is a real anchor — keyboard-reachable in tab order.
