Markup

<nav class="docs-breadcrumbs" aria-label="Breadcrumb">
<ol class="docs-breadcrumbs__list">
<li class="docs-breadcrumbs__item">
<a class="docs-breadcrumbs__link" href="/internal/">Internal</a>
</li>
<li class="docs-breadcrumbs__item">
<a class="docs-breadcrumbs__link" href="/internal/governance/">Governance</a>
</li>
<li class="docs-breadcrumbs__item" aria-current="page">Backlog</li>
</ol>
      </nav>

States

  • Default — link is muted (--muted); separator is a CSS-generated chevron.
  • Hover — link colour shifts to --text-strong; underline appears.
  • Focus — 2 px outline using --accent, offset 2 px.
  • Current — final item without <a>; uses aria-current="page"; rendered in --text with semibold weight.

A11y

  • Wrap in <nav aria-label="Breadcrumb"> so screen readers announce a navigation landmark.
  • Use an ordered list <ol> — order is meaningful.
  • Mark the current item with aria-current="page" and render it as plain text (no <a>).
  • Separators are decorative — generate via CSS ::before, not as visible text content.
  • Ensure colour contrast ≥ 4.5:1 for the link colour against --bg.

Tokens used

  • --text, --text-strong, --muted
  • --accent (focus ring)
  • --line-soft (separator chevron colour)
  • --fs-200, --fw-semibold
  • --space-2 (item gap)
