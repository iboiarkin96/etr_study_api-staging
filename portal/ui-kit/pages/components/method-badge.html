Method badge

Color-coded HTTP method labels for endpoint cards, reference tables, and inline API documentation. Also includes .docs-http-pill for HTTP status-code coloring.

Example

HTTP Methods

GET --get
POST --post
PUT --put
PATCH --patch
DELETE --delete
HEAD --head
OPTIONS --options

HTTP Status Pills

200 OK --2xx
301 --3xx
404 --4xx
503 --5xx

Accessibility

Badges are purely decorative labels. The surrounding context (path text, description) conveys the HTTP method to screen reader users. No additional aria-label is needed when the badge appears alongside a visible path string.

When used standalone in a table cell with no adjacent text, add an accessible name: <span class="docs-method-badge docs-method-badge--get" aria-label="GET method">GET</span>.

Tokens used

--success, --accent, --info, --warn, --danger, --muted, --radius-sm, --font-mono, --fs-100, --fw-semibold
