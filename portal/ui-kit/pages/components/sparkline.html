Sparkline

Compact SVG mini-chart for KPI cards and dashboards. Renders a polyline with gradient fill area. Pure SVG — no external libraries. Mount via data-component="sparkline".

Markup

html<!-- Default (accent, 80×32) -->
<span data-component="sparkline"
data-values="3,5,4,7,6,9,8,11"></span>

<!-- Custom color and size -->
<span data-component="sparkline"
data-values="28,31,35,38,36,40,38,42"
data-color="var(--success)"
data-width="100"
      data-height="40"></span>

Data attributes

AttributeDefaultDescription
data-values—Comma-separated numbers. Minimum 2 points required.
data-colorvar(--accent)CSS color or var() for line and fill gradient.
data-width80SVG width in pixels.
data-height32SVG height in pixels.

Accessibility

The rendered SVG has aria-hidden="true" — sparklines are decorative data visualizations. Accessible meaning should be conveyed by the adjacent KPI value and label, not the sparkline itself.

Tokens used

--accent, --success, --danger, --warn (any CSS color via data-color)
