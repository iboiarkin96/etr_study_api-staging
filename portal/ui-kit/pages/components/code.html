Code block

Syntax-highlighted code block with copy button and language tag. Used for snippets, terminal commands, and config examples.

Example

bash$ curl -sSf https://api.study-app.example/v1/health \
        -H "Authorization: Bearer $TOKEN" \
        | jq '.status'

Variants

Inline code

Use --space-3 for tight layouts and --space-5 for sections.

Markup

<pre class="docs-code" data-lang="ts">
<span class="docs-code__lang">ts</span>
<button class="docs-code__copy" type="button" aria-label="Copy code">Copy</button>
<code>export interface Session { id: string; }</code>
        </pre>

States

  • Default — copy button hidden until :hover or :focus-within.
  • Hover — copy button fades in.
  • Copied — JS sets data-state="copied"; button text becomes "Copied!" for 1.5 s.
  • Overflow — horizontal scroll inside the <pre>.

A11y

  • The copy button must have aria-label="Copy code".
  • After copying, announce success via aria-live="polite".
  • Keep the language tag readable for AT (do not aria-hidden if useful).
  • Ensure tab focus moves into the <pre> for keyboard horizontal scroll.

Tokens used

  • --font-mono, --inline-code-bg
  • --bg-mute, --line
  • --radius-md, --space-3
