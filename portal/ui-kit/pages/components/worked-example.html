Worked example

Bordered example block with a tone-tinted header (mono tag + title) and a content body. Use to show a copy-pasteable, realistic example next to abstract guidance — a YAML snippet beside ADR rules, a curl command beside a how-to step, a sample commit beside a process page.

Example

ADR 0042 · Move card content to row-level storage

Context. JSON blob limits queries; 30% of cards exceed 4 KB and search scans take 800 ms.

Decision. Migrate to one row per card field; keep the blob column for one release as a fallback.

-- 0001_card_fields.sql
CREATE TABLE card_fields (
  card_uuid  UUID NOT NULL REFERENCES cards(card_uuid),
  field_key  TEXT NOT NULL,
  field_val  TEXT NOT NULL,
  PRIMARY KEY (card_uuid, field_key)
            );

Consequences. Read path adds one join; write path becomes 3 statements; migration script lands in N follow-up PRs.

Anatomy

  • .pa-example — bordered outer wrapper, rounded, clipped.
  • .pa-example__head — tone-tinted header strip with a hairline bottom border.
  • .pa-example__tag — mono uppercase chip («Example», «Sample», «Live»).
  • .pa-example__title — short imperative title; semibold.
  • .pa-example__body — content area, --bg-elev background; children auto-space via * + *.
  • Inside the body, <pre> blocks get a neutral muted-bg look so code reads without clashing with the kit's global syntax-highlight theme.
