State strip

Horizontal pipeline of pill-shaped state badges separated by arrows, wrapped in a bordered box. Use to show a lifecycle, workflow, or pipeline at a glance — draft → proposed → accepted → superseded; stage 1 → stage 2 → stage 3. Each badge is tinted by --pa-accent (override per page).

Example

Draft → Proposed → Accepted → Superseded

Anatomy

  • .pa-states — bordered flex row with horizontal overflow on narrow viewports.
  • .pa-state — pill-shaped badge: tone-tinted background + border, mono text, no wrap.
  • .pa-state__arrow — separator between states. Use a literal → glyph.

Markup

<div class="pa-states">
  <span class="pa-state">Draft</span>
  <span class="pa-state__arrow">→</span>
  <span class="pa-state">Proposed</span>
  <span class="pa-state__arrow">→</span>
  <span class="pa-state">Accepted</span>
      </div>

Tone override

Override --pa-accent on the wrapper to re-tint the strip:

<div class="pa-states" style="--pa-accent: var(--teal)">
  …
      </div>

Naming note

Class names use the legacy .pa-states / .pa-state prefix; neutral rename to .state-strip is queued.
