Diagram lightbox

Zoomable, pannable lightbox for inline diagrams. The kit standardizes on PlantUML (Mermaid is forbidden per ADR 0020).

Markup

<!-- Lightbox shell -->
<div class="docs-diagram-lightbox" role="dialog" aria-modal="true" aria-label="Diagram viewer" hidden>
<button class="docs-diagram-lightbox__zoom-out" aria-label="Zoom out">−</button>
<button class="docs-diagram-lightbox__reset" aria-label="Reset zoom">⟳</button>
<button class="docs-diagram-lightbox__zoom-in" aria-label="Zoom in">+</button>
      </div>

States

  • Closed — only the inline trigger is visible.
  • Opening — backdrop fades in 160 ms; image scales from trigger bounds via FLIP.
  • Open — scroll-wheel zooms 1× → 8×; drag pans.
  • Zoomed — reset button enabled when scale ≠ 1.

A11y

  • Trigger must be a <button> with descriptive aria-label.
  • Lightbox uses role="dialog" aria-modal="true" and traps focus.
  • ESC closes; arrow keys pan when zoomed; + / − adjust zoom.
  • Each zoom button must have aria-label.
  • Provide a textual fallback caption in addition to the visual diagram.

Tokens used

  • --bg-elev, --shadow-lg, --radius-lg
  • --line-soft (toolbar separator)
  • --ease-out
