Theme toggle

Vertical-flashlight switch between light and dark, ported 1:1 from the legacy portal (top-nav__theme-btn--flashlight). Beam is dim in light mode and lit yellow with a warm halo in dark mode. Preference is persisted in localStorage["docs-theme-preference"] and bootstrapped from an inline <head> script to avoid a flash of the wrong theme.

Example

Click the flashlight below to toggle the theme.

Light → dark flips html[data-theme]. The cascade lights the beam.

Markup

<!-- 1. No-flash bootstrap (first <script> in <head>) -->
<script>
(function(){
try {
var v = localStorage.getItem("docs-theme-preference");
if (v === "dark" || v === "light") {
document.documentElement.setAttribute("data-theme", v);
}
} catch (e) {}
})();
</script>

<!-- 2. Empty button — theme-toggle.js renders the SVG flashlight -->
<button class="docs-theme-toggle" type="button" aria-label="Toggle theme"></button>

<!-- 3. Mount in entry.js -->
<script type="module">
import { mountThemeToggle } from ".../ui-kit/components/theme-toggle.js";
mountThemeToggle();
        </script>

States

  • Light — beam opacity 0.26, fill currentColor. Button border --line, color --muted. aria-pressed="false".
  • Dark — beam opacity 0.92, fill #fde047, drop-shadow 0 -2px 8px #fef08a. Button color #eab308, border tinted with #facc15, ambient halo via box-shadow. aria-pressed="true".
  • Hover (light) — border + color flip to --accent.
  • Hover (dark) — color goes to #facc15, border picks up 55 % of #fde047 blended with --accent.
  • Focus-visible — 2 px --accent outline, 2 px offset.

A11y

  • Always a real <button type="button"> — never an <a> or <div>.
  • aria-label="Toggle theme" on every instance; aria-pressed mirrors current state (true = dark / beam lit).
  • The flashlight SVG is aria-hidden="true" — the label carries the meaning.
  • Honour explicit user choice. The no-flash bootstrap reads localStorage; it does not override that with prefers-color-scheme at runtime.

Tokens used

  • --card, --line, --muted, --accent
  • Fixed lit-yellow palette: #eab308 / #facc15 / #fde047 / #fef08a
  • --ease-out
