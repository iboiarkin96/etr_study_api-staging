Status pill

Specialized pill for workflow states. Constrained vocabulary of five values — keeps backlogs and task lists consistent.

Example

todo in-progress done blocked rejected

Variants

ValuePillMeaning
todotodoCaptured, not yet scheduled.
in-progressin-progressActively being worked on.
donedoneAcceptance criteria met; closed.
blockedblockedCannot proceed; needs unblocking action.
rejectedrejectedDecided against; not moving forward.

States

Status pills are static. State is the variant — there are no hover / focus / disabled variants.

A11y

  • Always include the literal state text in the pill (no icon-only).
  • If the pill conveys real-time change, wrap the parent row in role="status" or aria-live="polite".
  • Maintain contrast ≥ 4.5:1 for the text against the tinted background.

Tokens used

  • --muted-soft (todo tint)
  • --info (in-progress)
  • --success (done)
  • --warn (blocked)
  • --rose (rejected)
  • --radius-full, --fs-100, --fw-semibold
