States

  • Row · default — zebra striping with --bg-mute on odd rows.
  • Row · hover — background tint 4% accent.
  • Cell · numeric — right-aligned via class .docs-table__num.
  • Sticky head — uses position: sticky; top: var(--layout-topbar) inside the wrap.

A11y

  • Every <th> needs scope="col" or scope="row".
  • Provide a <caption> for non-trivial tables.
  • Don't use tables for layout — only for tabular data.
  • The horizontal scroll wrap must be keyboard-scrollable (set tabindex="0").
  • Avoid colour as the only signal in status cells — pair with text.

Tokens used

  • --bg-elev, --bg-mute, --line, --line-soft
  • --fs-300
  • --space-2, --space-3 (cell padding)
