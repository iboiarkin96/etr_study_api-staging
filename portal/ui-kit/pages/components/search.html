States

  • Idle — placeholder visible.
  • Focused — accent ring, panel open if there are recents.
  • Typing — debounced 100 ms; results stream in.
  • Empty — no-results message.
  • Loading — subtle progress bar (1 px) inside the input border.

A11y

  • Outer element has role="search".
  • Input has aria-label if no visible label.
  • Panel has role="listbox"; each result role="option".
  • Use aria-activedescendant on the input to track keyboard selection without moving focus.
  • Arrow keys cycle results; Enter activates; ESC closes.
  • Announce result count via aria-live="polite".
