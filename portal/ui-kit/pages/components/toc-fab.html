TOC FAB

Floating action button that opens the TOC as a bottom-sheet drawer on mobile (≤1024 px). Visible only when at least two headings are present on the page.

Example

The real .docs-toc-fab is display:none at ≥1025px (it's a mobile-only FAB), so this preview uses inline styles to mirror the look.

Variants

  • Default — accent gradient background, white icon.
  • Open — icon rotates 90°, modifier .docs-toc-fab--open.
  • Hidden — set display: none via JS when fewer than two H2 sections.

Anatomy

One <button data-component="toc-fab"> placed anywhere in the page. mountTocFab() upgrades it, then:

  1. Decorates the trigger — adds .docs-toc-fab, injects the 24 × 24 list-lines icon, sets aria-label="Open table of contents" and a tooltip placed on the left.
  2. Builds the sheet panel — a <nav class="docs-toc-fab__panel"> body-portalled to the end of document.body, populated with every h2[id] / h3[id] found in main. If the page has zero headings, the FAB short-circuits and never renders the panel.
  3. Click-toggles the sheet — flips aria-hidden / aria-expanded; Esc closes; clicking outside the panel + FAB also closes.
  4. Idle dim — after 1.8 s without scroll, mouse, or touch input the FAB gains data-fab-idle="true" (CSS fades it). Any of those inputs wakes it.
  5. Sidebar-aware sticking — an IntersectionObserver watches the desktop sidebar. When the sidebar enters the viewport (open drawer on touch devices), the FAB gains data-fab-stuck="true" so styling can move it out of the way.

Markup

<button class="docs-toc-fab" type="button"
aria-label="Open on this page"
aria-controls="page-toc"
aria-expanded="false">
≡
        </button>

States

  • Default — solid gradient.
  • Hover — shadow upgrades to --shadow-lg.
  • Active — scale 0.94 with --ease-spring.
  • Open — icon flip + aria-expanded="true".

A11y

  • Must have an accessible name (aria-label).
  • Set aria-controls pointing at the TOC sheet's id.
  • When sheet is open, set aria-expanded="true"; trap focus inside the sheet.
  • Tap target ≥ 44 × 44 px.
  • Hide entirely on viewports ≥ 1025 px where the right-rail TOC is already visible.

Tokens used

  • --grad-accent, --shadow-md, --shadow-lg
  • --ease-spring
