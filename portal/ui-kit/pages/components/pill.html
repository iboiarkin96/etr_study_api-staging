States

Pills are static — no interactive states. Visual variation comes from the colour modifier only.

A11y

  • Pill colour must not be the only signal — pair with a text label.
  • Decorative dots get aria-hidden="true".
  • Maintain ≥ 4.5:1 text contrast on the tint background.
  • Do not use pills as the only way to convey error / warning severity in forms; pair with role="status" live region.

Tokens used

  • --success, --warn, --danger, --info, --rose, --teal, --muted
  • --radius-full
  • --fs-100, --fw-semibold
