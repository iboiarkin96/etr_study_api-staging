Radar

Two-polygon Now vs. Target chart for skills, role maturity, or any 0..N self-assessment with three or more axes. Renders an SVG plus a synced legend; hovering either side highlights the matching axis.

Example · Senior Software Architect ladder

Eight competencies, 0–5 ladder, current self-assessment vs. target

Compact variant · person card

Five axes, smaller size — fits inside a profile or person card. Hover the SVG or the rows to see the sync in action.

Markup

html<figure data-component="radar"
data-scale="5"
data-size="380"
data-title="SA competencies · self-assessment"
data-subtitle="Eight pillars from the senior-architect ladder."
data-caption="Snapshot · 2026-05-01"
data-axes='[
{"label":"System design","now":3.5,"target":5,"group":"A"},
{"label":"API contracts","now":4,  "target":5,"group":"A"},
{"label":"Data modeling","now":3,  "target":4,"group":"B"},
{"label":"Observability","now":2.5,"target":4,"group":"C"},
{"label":"Reliability",  "now":3,  "target":4.5,"group":"C"},
{"label":"Security",     "now":2,  "target":4,"group":"D"},
{"label":"Mentoring",    "now":3,  "target":4,"group":"E"},
{"label":"Stakeholders", "now":2.5,"target":4,"group":"E"}
          ]'></figure>

Data attributes

AttributeDefaultDescription
data-axes—JSON array of {label, now, target, group?, sub?}. Needs at least 3 axes; fewer renders nothing.
data-scale5Maximum value on each axis (integer ≥ 2). Five rings drawn from 1..scale.
data-size360SVG viewBox edge in px. Outer labels live inside the same box.
data-title—Heading shown above the legend.
data-subtitle—Helper text below the title.
data-caption—Caption below the SVG (e.g. snapshot date).
data-now-labelNowLegend chip text for the inner polygon.
data-target-labelTargetLegend chip text for the outer polygon.

Accessibility

Every legend row is a real <button> with an aria-label describing both scores, so the data is readable without the chart. The SVG carries role="img" and an aria-label. Polygons use both color and stroke pattern (solid for Now, dashed for Target) so the two surfaces remain distinguishable when color alone fails.

Tokens consumed

--accent, --rose (now polygon), --teal (target polygon), --info / --warn / --success (group badges), --text-strong, --muted, --line, --bg-elev, --space-*, --radius-*, --shadow-sm.
