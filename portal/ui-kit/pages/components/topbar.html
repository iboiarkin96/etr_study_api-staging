Topbar

Sticky page header. Holds the brand mark, breadcrumb slot, a spacer, then the actions cluster (search + theme toggle). Backdrop-blur over content as the user scrolls.

</div>
      </header>

States

  • Top of page — opaque background, no shadow.
  • Scrolled — translucent background with backdrop-blur, --shadow-sm applied.
  • Focus-within — when search has focus, actions cluster widens.

A11y

  • Use role="banner" on the outer element (HTML5 <header> implies it but be explicit when descendant of <main>).
  • Brand link must contain accessible text alongside the 🚀 (rocket alone is not a label).
  • The theme toggle must have aria-label and aria-pressed while toggled.
  • The search input needs an aria-label or visually-hidden label.
  • Sticky header must not exceed 56 px (--layout-topbar) on mobile to preserve viewport.

Tokens used

  • --layout-topbar (height)
  • --bg-elev, --line, --shadow-sm
  • --space-3, --space-4
  • --accent (brand text on hover)
