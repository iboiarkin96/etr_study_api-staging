Tokens

Design primitives consumed by every component. Tokens are declared as CSS custom properties on :root and on html[data-theme] / html[data-portal] for theme- and portal-scoped overrides.

Typography ladder

Inter for UI & prose, JetBrains Mono for code. Eight sizes (--fs-100 through --fs-900), uniform across desktop / tablet / mobile per the canonical font ladder.

--fs-900 · 2.75remDisplay heading
--fs-800 · 2remPage H1
--fs-700 · 1.5remSection H2
--fs-600 · 1.125remSubsection H3
--fs-500 · 1remLead paragraph
--fs-400 · 0.9remBody text — the canonical reading size used by paragraphs, lists, and most UI labels.
--fs-300 · 0.875remSecondary UI text and table cells.
--fs-200 · 0.8125remCaptions, badges, breadcrumb tail.
--fs-100 · 0.75remMicro labels & meta.

Line height & weight

--lh-tight 1.2 · --lh-snug 1.35 · --lh-base 1.55 · --lh-loose 1.75 — weights 400 / 500 / 600 / 700.

Motion · easing curves

Three named curves. All animations honor prefers-reduced-motion and reduce to ~0 ms when the user opts out.

--ease-out · cubic-bezier(0.22, 1, 0.36, 1)

--ease-in-out · cubic-bezier(0.45, 0, 0.55, 1)

--ease-spring · cubic-bezier(0.34, 1.56, 0.64, 1)

Portal accents

Each portal carries its own accent triplet via html[data-portal="…"]. Components reference --accent / --accent-2 / --accent-3 only — they remain portal-agnostic.

Gradients

The active portal's --grad-accent is shown below. This page is rendered with data-portal="internal", so you see the blue gradient.

html[data-portal="internal"] {
--accent: #2563eb;
--accent-2: #3b82f6;
--accent-3: #60a5fa;
--grad-accent: linear-gradient(135deg, #2563eb 0%, #3b82f6 45%, #60a5fa 100%);
}
