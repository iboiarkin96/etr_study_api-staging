# services/frontend/portal — Changelog

All notable changes to the **frontend tree** under `services/frontend/portal/` are tracked here. This journal is separate from [`services/portal/CHANGELOG.md`](../../portal/CHANGELOG.md) (documentation content) and the repository root [`CHANGELOG.md`](../../../CHANGELOG.md) (product and API behaviour). See [ADR 0013](../../portal/internal/governance/adr/0013-changelog-and-release-notes.html).

Scope:
- `assets_v2/` — canonical **UI Kit v2** (tokens, runtime entry, components).
- `assets/` — **legacy** kit (CSS/JS), the Pagefind search bundle, and the favicon. Being phased out per `services/frontend/portal/assets_v2/` migration.
- `uml/` — rendered PlantUML SVGs consumed by docs.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).
